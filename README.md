# Personal Website

My personal website built with React, TypeScript, and Vite, styled using Tailwind CSS.  
The design was created using Figma, but all the functionality and code were implemented manually.  

Live Site: https://vanessaching.github.io/personal-website/ 

---

# Overview

This website serves as a portfolio showcasing my skills, projects, and resume.

Sections include:

- About: Introduction and background
- Skills: Technical skills with interactive components
- Projects: Portfolio projects with descriptions and links
- Progress: Education and professional milestones
- Contact: Email and social links

Additional features:

- Smooth scrolling and animations
- Responsive design for desktop, tablet, and mobile

---

# Features

- Fully responsive design
- Custom React components with Radix UI
- Animations with Framer Motion
- Tailwind CSS for utility-first styling
- Resume and image assets integrated
- Accessible with semantic HTML and ARIA roles

---

# Tech Stack

- React 18 + TypeScript
- Vite for fast development
- Tailwind CSS for styling
- Framer Motion for animations
- Radix UI for accessible components
- GitHub Pages for deployment
